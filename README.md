## About Dataset

This dataset contains data of a store in MS Excel format. The columns in the dataset are as follows.

- Order ID
- Cust ID
- Gender
- Age
- Age Group
- Date
- Month
- Status
- Channel
- SKU
- Category
- Size
- Qty
- currency
- Amount
- ship-city
- ship-state
- ship-postal-code
